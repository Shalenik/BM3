# Plot Model Comparison

Create a plot comparing model performance across H values.

## Usage

``` r
plot_comparison(results, title = NULL)
```

## Arguments

- results:

  List from
  [`consolidate_results()`](https://shalenik.github.io/BM3/reference/consolidate_results.md).

- title:

  Optional plot title (default NULL).

## Value

ggplot2 object.
